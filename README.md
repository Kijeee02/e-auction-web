# Platform Lelang Barang Bekas

Platform lelang barang bekas terpercaya.

## Fitur
- Pencarian barang
- Pagination
- Validasi form login
- Loading indicator

## Cara Menjalankan
1. Clone repository ini.
2. Buka file `index.html` di browser.
3. Pastikan file JSON di folder `api/` dapat diakses.
